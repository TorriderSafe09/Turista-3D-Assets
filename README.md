# Turista-3D-Assets

Ecosistema y librería de activos (assets) en 3D optimizados para entornos virtuales y videojuegos, modelados en **Blender** con flujos de trabajo de texturizado de alta fidelidad.

## Estructura del Proyecto
* **`/.blend_files`**: Archivos fuente originales de modelado en alta precisión (Among, Billetes, Carro, Dados, Corona, X-Wing, Edificios, Tablero).
* **`/renders`**: Capturas de pantalla y previsualizaciones de los modelos terminados.
* **`/textures`**: Mapas de materiales PBR (Physically Based Rendering) incluyendo mapas de albedo, rugosidad (roughness), normales y desplazamiento (displacement) para realismo táctil (cuero, madera, concreto, roca).

## Especificaciones Técnicas
* **Herramienta principal**: Blender.
* **Técnicas utilizadas**: Modelado Hard Surface, control de flujo topológico (edge loops) y empaquetado de texturas.
* **Compatibilidad**: Diseñado para importación directa en motores de render y de videojuegos como Unity o Unreal Engine.

# Turista-3D-Assets 

Ecosystem and library of 3D assets optimized for virtual environments and video games, modeled in **Blender** with high-fidelity texturing workflows.

## Project Structure
* **`/.blend_files`**: Original high-precision modeling source files (Among, Bills, Car, Dice, Crown, X-Wing, Buildings, Board).
* **`/renders`**: Screenshots and previews of the finished models.
* **`/textures`**: PBR (Physically Based Rendering) material maps including albedo, roughness, normal, and displacement maps for tactile realism (leather, wood, concrete, rock).

## Technical Specifications
* **Main tool**: Blender.
* **Techniques used**: Hard Surface modeling, topological flow control (edge loops), and texture packing.
* **Compatibility**: Designed for direct import into rendering engines and game engines such as Unity or Unreal Engine.
